    Creates a painter object The argument passed during construction of this object determines the type of painter created.(param)

    Detailed Description

    Painter provides highly optimized functions to do most of the drawing GUI programs require. It can draw everything from simple lines to complex shapes like pies and chords. It can also draw aligned text and pixmaps. Normally, it draws in a "natural" coordinate system, but it can also do view and world transformation. Painter can operate on any object that inherits the Widget class.
